# 🎓 Student Record System

A C++ console application to manage student records.

## Features
- ✅ Add students
- ✅ Display all records
- ✅ Find topper
- ✅ Search by name
- ✅ Sort by marks
- ✅ Grade distribution
- ✅ Save/Load from file

## Tech Stack
- Language: C++
- Concepts: STL, File I/O, Sorting, Searching
- Tools: VS Code, Git, GitHub

## How to Run
```bash
g++ main.cpp -o student
./student
```
